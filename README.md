| Spec | Value |
|------|-------|
| Power | 3000W Continuous / 5kW Peak |
| Voltage | 14-58V (14S battery support) |
| Control | FOC with SVPWM at 20-50 kHz |
| Feedback | Dual encoder, 24-bit absolute |
| Interface | Isolated CAN/UART, ROS2 |

**Vector Control Pipeline:**
```
Encoder → FOC (Clarke/Park) → SVPWM → BLDC Motor
         ↓
    Id (flux) | Iq (torque) decoupled control
```
